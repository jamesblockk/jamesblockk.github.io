使用docker 開始 rocket chat!!

Posted by JamsBlocKK Blog on Wednesday, February 20, 2019

rocket chat start

測試環境 : macOS

reference

源碼

＄git clone https://github.com/RocketChat/Rocket.Chat
＄cd Rocket.Chat

通過以下方式啟動mongodb服務器：

$ docker-compose up -d mongo

第一次啟動mongo時，你還需要初始化它才能使用Rocket.Chat。確保mongo處於運行狀態，然後：

$docker-compose up -d mongo-init-replica

Mongo支持24 x 7操作和實時備份。您不需要太頻繁地重新啟動它。有關mongo服務器的正確操作和管理，請參閱 mongodb文檔。

一旦你確定mongodb啟動並運行：

$ docker-compose up -d rocketchat

在瀏覽器中進入 localhost:3000 就可以看到畫面

mongo

進入 mongodb 中

$docker exec -it rocketchat_mongo_1 mongo  --authenticationDatabase admin

查看 db

mongo> show dbs
mongo> use rocketchat
mongo> show collections
