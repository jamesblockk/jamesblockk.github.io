進入容器

在使用 -d 參數時，容器啟動後會進入背景執行

進入容器 :

$sudo docker exec -ti (image_name) bash

進入容器後.需要編輯文件, 編輯器的安裝方式 :

$root@62697b2bdacf:/ apt-get update
$root@62697b2bdacf:/ apt-get -y install vim
